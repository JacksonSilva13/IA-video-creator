# IA Video Creator

Aplicativo móvel para criar vídeos curtos com inteligência artificial.

## Recursos

- ✨ Geração automática de roteiros com IA
- 💰 Pagamento via Pix (R$ 19,90)
- 🎬 3 vídeos gratuitos por mês
- 📱 Interface moderna e intuitiva
- 🎨 Sem marca d'água (Premium)

## Como Usar

1. Descreva sua ideia de vídeo
2. A IA gera um roteiro completo
3. Personalize conforme necessário
4. Crie e compartilhe seu vídeo
